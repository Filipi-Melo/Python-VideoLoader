# Python-VideoLoader
# Descrição
- Um leve aplicativo baixador de videos do Youtube.
- Baixe videos, músicas, thumbs e playlists rapidamente.
- Possui uma interface interativa.

# Instalação
- Baixe o executável
- Descompacte o arquivo
- Ponha o arquivo na área de trabalho
